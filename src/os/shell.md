# Shell

## TL;DR

운영 체제 기능과 서비스를 구현하는 인터페이스를 제공하는 프로그램이다. **사용자**와 **운영 체제의 커널** 사이의 인터페이스를 감싸는 프로그램이기 때문에 셸이란 이름이 붙었다.

## 셸의 목록

### CLI

* Unix
  * sh - ash, bash
  * csh - tsch
  * ksh
  * zsh
* COMMAND.COM: MS-DOS 용
* CMD.EXE: Windows NT 용

### GUI

* 윈도 탐색기: MS Windows 계열
* 매킨토시 파인더
* X 윈도 시스템
