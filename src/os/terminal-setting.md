# 터미널 환경 세팅

## zsh 설치

1\. macOS

```sh
> brew install zsh # zsh을 설치
```

2\. linux

```sh
> apt-get install zsh # zsh을 설치
```

## 기본 셸 변경 

```sh
> which zsh # 셸의 위치를 확인
> chsh -s /usr/bin/zsh # 기본 셸을 zsh로 변경
```

## `oh-my-zsh` 설치

```sh
> curl -L https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh | sh
```
