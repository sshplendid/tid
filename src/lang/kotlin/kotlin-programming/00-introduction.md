## 특징

* Java 100% 호환
* JVM 기반 언어(Run everywhere)
* Java/Kotlin 혼용 가능
* 개발 생산성: Java 10줄짜리 코드가 코틀린 2줄로...
* 함수형 프로그래밍, Null safety 등 언어 트렌드 지원
* 크로스 플랫폼 개발
  * Server-side 개발 가능(Spring 5부터 코틀린 지원)
  * Kotlin.js: 브라우져 개발
  * Kotlin/Native (v1.3부터): without JVM
* Google의 (전폭적인) 지원
